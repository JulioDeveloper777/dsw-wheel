# DSW-Wheel
A Modern Platform for DIY Steering Wheel.

## Technologies
- **Frontend**
- Next.js
- TypeScript
- Tailwind CSS

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```
4. Open your browser and visit http://localhost:3000 to see the app.
